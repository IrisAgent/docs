## **Getting Started**

### **Step 1: Install IrisGPT in Your Slack Workspace**

Install the IrisGPT bot using the install link provided by your IrisAgent point of contact. When prompted by Slack, authorize the app to access your workspace.

No additional setup is required — your IrisAgent contact handles all app configuration on your behalf.

### **Step 2: Choose a Deployment Mode**

IrisGPT supports two deployment modes. Discuss your preferred mode with your IrisAgent contact during setup.

#### **Option 1: Channel Auto-Reply Mode**

IrisGPT automatically responds to messages in the channels where it is added. This mode is ideal for dedicated support or help channels where you want every question answered.

#### **Option 2: Mention Mode**

IrisGPT only responds when explicitly mentioned with `@IrisGPT`. This mode is ideal for general channels where you want on-demand assistance without automatic replies.

### **Step 3: Invite IrisGPT to Your Channels**

To add IrisGPT to a channel, type the following command in that channel:

```
/invite @IrisGPT
```

The bot must be invited to a channel before it can respond to messages there.

## **Using IrisGPT in Slack**

### **Slash Command: `/askIrisGpt`**

Use the slash command to ask a question privately. The response is visible only to you.

```
/askIrisGpt how do I reset my password?
```

### **Mention: `@IrisGPT`**

Mention the bot in any channel where it has been invited. The response is posted as a public thread reply, visible to everyone in the channel.

```
@IrisGPT how do I reset my password?
```
